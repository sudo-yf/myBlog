# 欢迎！

这是你的第一篇博客文章。你可以使用 Markdown 和 MDX 语法来编写内容。

## 功能特性

- **Markdown 支持**：使用标准 Markdown 语法
- **代码高亮**：支持多种编程语言的语法高亮
- **数学公式**：支持 LaTeX 数学公式
- **图片支持**：可以插入本地或在线图片

## 如何编写文章

1. 在 `src/content/blog/` 目录下创建新的 `.md` 或 `.mdx` 文件
2. 添加 frontmatter 元数据（标题、描述、日期等）
3. 使用 Markdown 编写正文内容
4. 保存文件，开发服务器会自动刷新

  console.log('Hello, World!');
